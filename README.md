Quest-Based Task Manager:- Seena Patel 25CS078


A simple CLI-based task manager that turns daily tasks into **quests** using a gamified system with XP and rewards.


This project is a console-based application written in C++ that helps users manage tasks in a fun and interactive way.
Each task is treated as a *quest*, and completing quests earns XP, which can be used to unlock rewards.

Features:

Add new quests
View all quests
Mark quests as completed
 Delete quests
 Earn XP for completing tasks
 Reward system (redeem using XP)
  user-friendly 


 Compile the program

```bash
g++ questbased.cpp -o quest
```

 Run the program

```bash
./quest
```
